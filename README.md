# Random-Password-Generator-Python
Project Overview

The Random Password Generator is a Python application that generates strong and secure passwords based on user-defined length. It uses Python's built-in modules to create random combinations of letters, numbers, and special characters.

Objective

To develop a secure password generation tool using Python and demonstrate the use of modules, string manipulation, and secure randomization techniques.

Technologies Used

- Python 3
- Secrets Module
- String Module

Features

- User-defined password length
- Secure random password generation
- Includes uppercase and lowercase letters
- Includes digits and special characters
- Simple and easy-to-use interface

How to Run

1. Clone the repository.
2. Open the project folder.
3. Run the Python file:

python password_generator.py

4. Enter the desired password length.
5. The program will generate a secure password.

Sample Output

Enter password length: 10

Generated Secure Password: A@7m#P2xL$
